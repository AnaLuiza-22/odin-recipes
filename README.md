# Odin Recipes

This is a project with The Odin Project Foundation Course.
The goal is to create a basic wesbite using HTML.

## Skills Demonstrated

- Writing clear and structured HTML
- Using semantic elements
- Creating links between pages
- Organising files and folders
- Adding images with alt text and figure captions
- Using Git for version control
- Deploying a website with Github Pages
